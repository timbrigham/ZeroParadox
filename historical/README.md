## Viewing Development History

For the complete commit history and detailed changes over time, use Git:

- `git log --oneline` — See all commits
- `git show <commit-hash>:filename` — View a file's content at a specific commit
- `git log -p filename` — See all changes made to a specific file

See the main [README](../README.md) for more details on accessing version history.
